
# Chat por consola (sockets puros)

Este folder contiene las fases originales del proyecto **secure-chat** en Python usando `socket` y `threading`.

## Archivos
- `server_step1.py`: servidor básico (eco/broadcast simple).
- `client_step1.py`: cliente básico.
- `server_step2.py`: servidor con manejo de múltiples clientes y nombres.
- `client_step2.py`: cliente con nombre de usuario.

## Cómo ejecutar (ejemplo step2)
1. En una terminal:
   ```bash
   python server_step2.py
   ```
2. En otra(s) terminal(es):
   ```bash
   python client_step2.py
   ```
3. Seguí las instrucciones en pantalla.
